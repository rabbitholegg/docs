# Boost Action Create Guide

This guide will help you find the right information on Etherscan to set up the action for your boost. Remember, choosing the action is about defining what on-chain behavior you want to incentivize. Setting the reward amount and distribution is a separate step that comes after you've defined the action.

## What You'll Learn

In this guide, you'll learn how to:

1. Navigate Etherscan to find relevant transaction information
2. Identify key components of a transaction for creating a boost action
3. Set up effective filters for your boost
4. Understand the difference between action definition and reward setup

Let's get started with the basics of using Etherscan to gather the information you need.
